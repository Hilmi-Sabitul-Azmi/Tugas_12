# Sistem Manajemen Nilai Mahasiswa

Program sederhana berbasis Python untuk mengelola data nilai mahasiswa
menggunakan File Handling (CSV).

## Fitur
- Input data mahasiswa
- Menampilkan data nilai
- Mencari data berdasarkan nama

## Teknologi
- Python
- CSV File
- Library tabulate

## Cara Menjalankan
1. Pastikan Python sudah terinstall
2. Install library tabulate:
   pip install tabulate
3. Jalankan program:
   run python file
